![图 0](img/4118ee5257922359a50f0c05ee459dd7b4ce47d835c322de26da6bf89fa8fc4e.png)  
浏览器会首先加载HTML，转换为dom树，然后加载css文件（css文件中的样式表会根据选择器的不同，放到不同的桶中），最后浏览器基于它找到不同的选择器。
 >当遇到无法解析的css代码，浏览器会忽略这一行。且浏览器会加载样式表中最后的css代码进行渲染。
